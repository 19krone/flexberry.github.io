Разработчик конструирует любой объект данных стандартным для `.Net` образом: вызовом `new` с необходимым конструктором.

Для нашего примера:

```csharp
SimpleDataObject sdo = new SimpleDataObject();
sdo.Master = new MasterDataObject();
```
