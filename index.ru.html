Продукты платформы Flexberry

Flexberry Designer

Инструмент проектирования и генерации кода. Поддержка UML.

ember-flexberry

Frontend framework для создания амбициозных приложений. Всё что нужно для организации форм ввода и поиска данных. Основн на EmberJS.

ember-flexberry-data

Frontend framework предоставляющий слой доступа к данным. Поддерка OData, проекций, offline, языка запросов. Основан на EmberJS (Ember.Data).

Flexberry ORM

ORM для .NET Framework. Благодаря расширенной поддержке проекций позволяет эффективно и гибко обрабатывать только нужные в данный момент данные.

Flexberry ODataService

OData v4 интерфейс для Flexberry ORM. Гибкая настройка, удобное использование.

ember-flexberry-security

Frontend framework для организации разделения полномочий пользователей и аудита изменений данных.

ember-flexberry-gis

Frontend framework для реализации ГИС-приложений на базе ember-flexberry.

Leaflet-WFST

Плагин для компонента Leaflet, реализующий протокол WFS-T.

Leaflet.Editable.Measures

Плагин для компонента Leaflet, реализующий инструменты изменения.
