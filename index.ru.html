Компоненты платформы Flexberry

Flexberry Designer

Инструмент проектирования и генерации кода (нотация UML).

Leaflet-WFST

Плагин для компонента Leaflet, реализующий протокол WFS-T.

ember-flexberry-data

EmberJS addon, реализующий слой доступа к данным. Поддерживает OData, проекции, offline, язык запросов.

ember-flexberry

EmberJS addon для создания амбициозных приложений. Всё что нужно для организации форм ввода и поиска данных.

Flexberry ODataService

OData v4 backend на основе Flexberry ORM. Гибкая настройка, удобное использование.

Flexberry ORM

ORM с расширенной поддержкой проекций и различных типов СУБД.

Flexberry ASP.NET

Фреймворк для создания сложных ASP.NET WebForms приложений.

ember-flexberry-security

EmberJS addon для разделения прав пользователей и аудита изменения данных.

ember-flexberry-gis

EmberJS addon для реализации ГИС-приложений на базе ember-flexberry.

Leaflet.Editable.Measures

Плагин для компонента Leaflet, реализующий инструменты изменения.

ember-flexberry-gis-yandex

EmberJS addon для поддержки API Яндекс.Карт на базе ember-flexberry-gis.

ember-flexberry-gis-esri

EmberJS addon для поддержки API ESRI ArcGIS на базе ember-flexberry-gis.
