    ListFormRoute Class

    Uses
    • PaginatedRouteMixin
    • SortableRouteMixin
    • LimitedRouteMixin
    Extends ProjectedModelFormRoute
    Defined in: ember-flexberry/addon/routes/list-form.js:12
    Module: ember-flexberry

    Base route for the List Forms.

    This class re-exports to the application as /routes/list-form. So, you can inherit from ./list-form, even if file app/routes/list-form.js is not presented in the application.

    Example:

    // app/routes/employees.js
import ListFormRoute from './list-form';
export default ListFormRoute.extend({
});

    If you want to add some common logic on all List Forms, you can define (actually override) app/routes/list-form.js as follows:

    // app/routes/list-form.js
import ListFormRoute from 'ember-flexberry/routes/list-form';
export default ListFormRoute.extend({
});
